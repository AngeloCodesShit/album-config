# Configuratore Album - Studio Andrea Materia

## Come aggiornare
Per modificare prezzi o prodotti, apri il file `index.html` con un editor di testo e modifica la sezione iniziale del codice Javascript sotto "DATI E CONFIGURAZIONE".

## Come pubblicare
1. Carica il file `index.html` su GitHub.
2. Vai in Settings > Pages.
3. Seleziona Source: "Deploy from a branch" -> "main" -> "/ (root)".
